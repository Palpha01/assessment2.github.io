Welcome to the For Honor Website, an encyclopedia about Ubisoft's For Honor. For Honor is a third-person action game with singleplayer and multiplayer modes in which warriors from five main groups fight each other: the Knights, the Samurai, the Vikings, the Wu Lin, and the Outlanders. The website serves a repository of information about the game, including the campaign, classes, multiplayer, and maps.
